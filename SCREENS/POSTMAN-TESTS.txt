 Postman Tests

1️⃣ GET all tasks

```
GET
http://localhost:5000/tasks
```

2️⃣ GET task by ID

```
GET
http://localhost:5000/tasks/1
```

3️⃣ POST - Create new task

```
POST
http://localhost:5000/tasks
Body (JSON):
{
    "title": "New task"
}
```

4️⃣ PUT - Update task

```
PUT
http://localhost:5000/tasks/2
Body (JSON):
{
    "title": "Build a Task API - UPD"
}
```

5️⃣ DELETE - Remove task

```
DELETE
http://localhost:5000/tasks/1
```
